Compiling on Linux:
Dependencies:
	libsdl2-dev
	libsdl2-mixer-dev
	libsdl2-image-dev
Run make

Compiling on Windows:
Run "make windows"

Controls:
Left, Right - Move left, right
Z, X - Rotate clockwise, counter clockwise
Down - Speed up drop
Space - Hard drop

F5 - debug mode - In debug mode you move with arrows on press
